# 已知问题 

1. 和系统设置"自动切换到文稿的输入法"可能冲突，使用GhostSKB时，请把"自动切换到文稿的输入法"取消勾选。 设置->键盘->输入法-> "自动..."
2. GhostSKB对大多数软件有效。如果某软件自身带了输入法自动切换的功能，或者应用软件在来回切换的时候不发系统消息通知，那么GhostSKB就没有效果
