# GhostSKB

一个自动切换输入法的工具
![logo of ghostkbd](https://github.com/dingmingxin/GhostSKB/blob/master/Resources/ghostkbd-256.png)

# 系统要求

Mac OS X 10.10 or above

# 特点

- [x] 1. 自动根据应用切换输入法
- [x] 2. icloud同步配置文件
- [x] 3. 支持多个profile
- [x] 4. 每个输入法可以设置一个快捷键
